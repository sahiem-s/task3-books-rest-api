# task3-books-rest-api
 Books REST API (Node.js + Express) | A simple REST API that manages a list of books using in-memory storage. Endpoints included: GET /books (get all books), GET /books/:id (get one book), POST /books (add new book), PUT /books/:id (update book), DELETE /books/:id (delete book). Setup: 1) Run npm install 2) Start server with node index.js 3) API runs at http://localhost:3000. Use Postman to test: send JSON body like {"title":"Book Name","author":"Author Name"} for POST/PUT.
